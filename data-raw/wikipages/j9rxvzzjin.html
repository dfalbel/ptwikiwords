ALT Linux é um conjunto de sistemas operacionais baseados no Linux. O ALT Linux é desenvolvido em conjunto pela comunidade de desenvolvedores do ALT Linux e a equipe da ALT Linux Ltd.[3][4] É certificado pelo Federal Service for Technical and Export Control (FSTEC Russia).[5]

História[editar | editar código-fonte]

Em 1999-2000, a distribuição desenvolveu o núcleo da futura ALT Linux Team, com base na distribuição MandrakeLinux aonde era uma versão russa (Mandrake Russo).

Desde 2000, começou a produzir pacotes Mandrake, uma significativa mudança no sistema em assembler e no gerenciador de pacotes RPM. De acordo com a versão 3,0 (2005) todos os pacotes Mandrake Russo, instalados no sistema, foram completamente produzidos pela própria ALT Linux Team. Agora as distribuições ALT Linux estão em seguindo um ramo distinto e não tem nada a ver com o Mandrake ou Mandriva.

Distribuições[editar | editar código-fonte]

Lite[editar | editar código-fonte]

ALT Linux Lite - Uma pequena distribuição para computadores mais antigos e de pouca de memória, com o Xfce como desktop. Disponível nas versões Normal e Live CD

Compact[editar | editar código-fonte]

ALT Linux Compact - distribuições sob medida para usuários iniciantes. É suposto ser utilizado principalmente em estações de trabalho, casa computadores e notebooks. Inclui características adicionais para uma fácil configuração, além de todas as funções e aplicações multimídia, e alguns jogos. Compact também é popular escolha para OEMing, ou seja, criando uma edição específica apenas incluindo os drivers do seu hardware.

A última versão do Linux ALT Compact (3.0) foi lançada em dezembro de 2003 (CD, DVD e Live CD). A distribuição Compact já foi descontinuada e não possui mais atualizações.

Desktop[editar | editar código-fonte]

ALT Linux Desktop - distribuição certificada pela FSTEK Rússia[6]

Office Server[editar | editar código-fonte]

ALT Linux Office Server - distribuição com um conjunto fixo de elementos servidor, totalmente personalizável e com interface web.[7]

Server[editar | editar código-fonte]

ALT Linux 4.0 Server [8] - distribuição servidor certificada pela FSTEK Rússia[9] com administração via web e atualização constante.

Terminal[editar | editar código-fonte]

ALT Linux 4.0 Terminal - distribuição para a implantação de Terminal Server/Client.

A Distribuição apresenta baixos requisitos de sistema para computadores que são usados como terminais; A distribuição está desenvolvendo uma empresa parceira de Kiev chamda "Media Magic";

Children (Criança)[editar | editar código-fonte]

A distribuição é destinada a crianças para desenvolver a criatividade ou apara professores no campo da computação gráfica e vídeo digital. Base de distribuição é um curso de formação gráfica, animação, vídeo, juntamente com ferramentas de software multimídia e materiais necessários para a realização das aulas. ALT Linux 4.1 Criança também inclui o famoso GCompris Suite para crianças com menos de 10 anos de idade, e outros materiais de referência, e uma pequena seleção de jogos de computador.[10]

Atualmente a Versão Children 4.1 (Criança) está em Fase Beta Live CD (O principal curso de formação e materiais estão em russo apenas e o KDE, GCompris e jogos estão em Inglês, em sua versão Oficial estará disponível em outras linguagens )
