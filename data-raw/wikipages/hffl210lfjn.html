A integral de uma função positiva pode ser interpretada como a área sob a curva de um gráfico.

Em matemática a integral de Lebesgue é uma generalização da integral de Riemann. Originalmente definida para funções f : R n → R {\displaystyle f:\mathbb {R} ^{n}\to \mathbb {R} } , a integral de Lebesgue apresenta diversas vantagens em relação à integral de Riemann sobretudo em relação a processos de limite. De fato, não existem versões dos teorema da convergência monótona, teorema da convergência dominada e do lema de Fatou usando a integral de Riemann. Além disso, a integral de Lebesgue é uma construção matemática generalizável para funções definidas num espaço de medida assumindo valores reais ou complexos, ou mesmo, em um espaço de Banach geral.

Construção[editar | editar código-fonte]

Existem diversas possíveis construções para integral de Lebesgue, seguiremos aqui um método baseado na exaustão por funções simples.

Considere, então, ( X , M , μ ) {\displaystyle \left(X,{\mathfrak {M}},\mu \right)\,} um espaço de medida.

Funções simples[editar | editar código-fonte]

Seja ϕ : E → [ − ∞ , + ∞ ] {\displaystyle \phi :E\to [-\infty ,+\infty ]\,} uma função simples:

ϕ ( x ) = ∑ k = 1 n α k X E k ( x ) {\displaystyle \phi (x)=\sum _{k=1}^{n}\alpha _{k}\mathrm {X} _{E_{k}}(x)\,}

Diz-se que ϕ ( x ) {\displaystyle \phi (x)\,} é Lebesgue integrável em E {\displaystyle E\,} se:

∑ k = 1 n | α k | μ ( E k ) < ∞ {\displaystyle \sum _{k=1}^{n}|\alpha _{k}|\mu (E_{k})<\infty \,} ficando bem convencionado que + ∞ ⋅ 0 = − ∞ ⋅ 0 = 0 ⋅ + ∞ = 0 ⋅ − ∞ = 0 {\displaystyle +\infty \cdot 0=-\infty \cdot 0=0\cdot +\infty =0\cdot -\infty =0\,}

neste caso, definimos a integral de Lesbesgue de ϕ ( x ) {\displaystyle \phi (x)\,} como:

∫ E ϕ ( x ) d μ = ∑ k = 1 n α k μ ( E k ) {\displaystyle \int _{E}\phi (x)d\mu =\sum _{k=1}^{n}\alpha _{k}\mu (E_{k})\,}

Funções positivas[editar | editar código-fonte]

Seja f : E → [ 0 , + ∞ ] {\displaystyle f:E\to [0,+\infty ]\,} uma função mensurável, define-se a integral de Lebesgue de f ( x ) {\displaystyle f(x)\,} em E {\displaystyle E\,} como:

∫ E f ( x ) d μ = sup ϕ ( x ) ≤ f ( x ) ∫ E ϕ ( x ) d μ {\displaystyle \int _{E}f(x)d\mu =\sup _{\phi (x)\leq f(x)}\int _{E}\phi (x)d\mu \,} , onde ϕ ( x ) {\displaystyle \phi (x)\,} é uma função simples.

A função f ( x ) {\displaystyle f(x)\,} é dita, então, Lebesgue integrável se sua integral é finita. Observações:

  • Quando f ( x ) {\displaystyle f(x)\,} é uma função simples, esta definição é consistente com a definição anterior.
  • A integral de Lebesgue está definida para toda função mensurável não negativa. A integral sendo finita se e somente se a função é Lebesgue integrável.

Funções reais[editar | editar código-fonte]

Seja f : E → [ − ∞ , + ∞ ] {\displaystyle f:E\to [-\infty ,+\infty ]\,} uma função mensurável, definem-se as partes positivas e negativas, respectivamente como:

f + ( x ) = { f ( x ) , f ( x ) ≥ 0 0 , f ( x ) ≤ 0 {\displaystyle f^{+}(x)=\left\{{\begin{array}{ll}f(x),&f(x)\geq 0\\0,&f(x)\leq 0\end{array}}\right.\,}
f − ( x ) = { 0 , f ( x ) ≥ 0 − f ( x ) , f ( x ) ≤ 0 {\displaystyle f^{-}(x)=\left\{{\begin{array}{ll}0,&f(x)\geq 0\\-f(x),&f(x)\leq 0\end{array}}\right.\,}

É fácil ver que se f ( x ) {\displaystyle f(x)\,} é mensurável, então ambas f + {\displaystyle f^{+}\,} e f − {\displaystyle f^{-}\,} são mensuráveis não negativas e que f ( x ) = f + ( x ) − f − ( x ) {\displaystyle f(x)=f^{+}(x)-f^{-}(x)\,} .

A função f ( x ) {\displaystyle f(x)\,} é dita Lebesgue integrável em E {\displaystyle E\,} se ambas as integrais ∫ E f + ( x ) d x {\displaystyle \int _{E}f^{+}(x)dx\,} e ∫ E f − ( x ) d x {\displaystyle \int _{E}f^{-}(x)dx\,} forem finitas e sua integral é definida como:

∫ E f ( x ) d x = ∫ E f + ( x ) d x − ∫ E f − ( x ) d x {\displaystyle \int _{E}f(x)dx=\int _{E}f^{+}(x)dx-\int _{E}f^{-}(x)dx\,}
  • Observe que f ( x ) {\displaystyle f(x)\,} é integrável se e somente se | f ( x ) | {\displaystyle |f(x)|\,} é integrável.

Propriedades[editar | editar código-fonte]

Se f ( x ) {\displaystyle f(x)\,} e g ( x ) {\displaystyle g(x)\,} são funções integráveis em um conjunto mensurável E {\displaystyle E\,} , então:

  • ∫ E [ α f ( x ) + β g ( x ) ] d μ = α ∫ E f ( x ) d μ + β ∫ E g ( x ) d μ {\displaystyle \int _{E}\left[\alpha f(x)+\beta g(x)\right]d\mu =\alpha \int _{E}f(x)d\mu +\beta \int _{E}g(x)d\mu \,}
  • f ( x ) ≤ g ( x ) {\displaystyle f(x)\leq g(x)\,} quase sempre, então ∫ E f ( x ) d μ ≤ ∫ E g ( x ) d μ {\displaystyle \int _{E}f(x)d\mu \leq \int _{E}g(x)d\mu \,}
  • | ∫ E f ( x ) d μ | ≤ ∫ E | f ( x ) | d μ {\displaystyle \left|\int _{E}f(x)d\mu \right|\leq \int _{E}|f(x)|d\mu \,}
  • F ⊆ E {\displaystyle F\subseteq E\,} mensurável, f ( x ) {\displaystyle f(x)\,} é integrável em F {\displaystyle F\,} e, ainda:
∫ E f ( x ) d μ = ∫ F f ( x ) d μ + ∫ E ∖ F f ( x ) d μ {\displaystyle \int _{E}f(x)d\mu =\int _{F}f(x)d\mu +\int _{E\backslash F}f(x)d\mu \,}
  • Se { E n } n = 1 ∞ {\displaystyle \{E_{n}\}_{n=1}^{\infty }\,} são subconjuntos mensuráveis e disjuntos dois a dois e ⋃ n = 1 ∞ E n = E {\displaystyle \bigcup _{n=1}^{\infty }E_{n}=E\,} então:
∫ E f ( x ) d μ = ∑ n = 1 ∞ ∫ E n f ( x ) d μ {\displaystyle \int _{E}f(x)d\mu =\sum _{n=1}^{\infty }\int _{E_{n}}f(x)d\mu \,}
  • ν ( F ) := ∫ F f ( x ) d μ {\displaystyle \nu (F):=\int _{F}f(x)d\mu \,} define uma medida nos subconjuntos mensuráveis de E {\displaystyle E\,} .

Comparação com a integral de Riemann[editar | editar código-fonte]

  • A integral de Riemann no sentido próprio só está definida em intervalos finitos ou na união finita destes. Se uma função é integrável a Riemann em um intervalo [ a , b ] {\displaystyle [a,b]\,} então a integral de Lebesgue também está definida e possui o mesmo valor.
  • Enquanto toda função integrável a Riemann é limitada, existem funções integráveis a Lebesgue que não são limitadas nem mesmo essencialmente limitadas em nenhum aberto do domínio.
  • O domínio de integração da integral de Lebesgue pode ser qualquer conjunto mensurável, inclusive não limitado.
