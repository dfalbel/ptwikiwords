Apache XMLBeans

Origem: Wikipédia, a enciclopédia livre.
Ir para: navegação, pesquisa

XMLBeans é um framework para binding de XML com Java. Ele é parte do projeto XML da Apache Software Foundation.

O framework XMLBeans permite que um programador da linguagem Java acesse informações em XML através de objetos Java. Esses objetos são obtidos de classes geradas pela pela ferramenta e que mapeiam diretamente a hierarquia de um determinado documento XML. Isto pode ser feito das seguintes maneiras:

  • através de um esquema XML que é utilizado para gerar as classes Java necessárias. Desta maneira o programador interage com objetos no estilo JavaBean que representam a estrutura do XML;
  • um cursor que navega por toda a hierarquia do XML;
  • suporte às APIs DOM do XML.
