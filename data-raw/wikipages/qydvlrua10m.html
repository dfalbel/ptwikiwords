Google Web Server

Origem: Wikipédia, a enciclopédia livre.
Ir para: navegação, pesquisa

Google Web Server (GWS) é o nome do servidor web que o Google utiliza para a sua infraestrutura. Foi confirmado que trata-se de uma compilação do Apache Web Server com mudanças no cabeçalho.[1] Sua última versão é a 2.1.
